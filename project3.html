UniSphere Application Security

A project at BML Munjal University focused on enhancing the security of the UniSphere application by implementing robust database access controls.

Project Overview

In my role as a Cyber Security Analyst for the UniSphere application, I coordinated directly with the database security department to identify and mitigate potential vulnerabilities. The primary objective was to secure user data and enforce strict permissions within the application's backend.

My main responsibility was the implementation of a comprehensive access control feature using Firebase's Security Rules language. This involved writing a complex ruleset to define which users could read, write, or update specific documents and collections in the Firestore database.

The rules were designed to be both secure and scalable, ensuring that as the application grew, the data remained protected against unauthorized access. This initiative significantly hardened the application's security posture and protected sensitive user information.
